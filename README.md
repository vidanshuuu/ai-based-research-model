# Autonomous Research Agent (LangChain)

## 📌 Objective
Build an AI system that can automatically research a topic and generate a structured report.

---

## ⚙️ Features
- Web Search using DuckDuckGo
- Knowledge Extraction using Wikipedia
- Automated Research Pipeline
- Structured Report Generation

---

## 🛠️ Technologies Used
- Python
- LangChain
- HuggingFace Transformers

---

## 🧠 How It Works
1. Takes a topic as input
2. Searches web data using DuckDuckGo
3. Fetches knowledge from Wikipedia
4. Combines and processes data
5. Generates a structured report

---
